# 使用说明

## 一、打开上传页

访问：

```text
https://ayayadaze.github.io/ImageHost/upload.html
```

## 二、填写 Worker 上传接口

部署 Cloudflare Worker 后，会得到一个地址，例如：

```text
https://imagehost-uploader.xxx.workers.dev
```

上传接口需要填写：

```text
https://imagehost-uploader.xxx.workers.dev/upload
```

浏览器会自动记住这个地址，下次不用重复填写。

## 三、上传图片

1. 把图片拖入上传区域，或者点击上传区域选择图片。
2. 支持多选。
3. 点击 `上传并生成 URL`。
4. 成功后页面会显示：
   - 直接 URL
   - Markdown 写法

Markdown 示例：

```text
![image](https://ayayadaze.github.io/ImageHost/images/2026/05/example.png)
```

## 四、支持格式

```text
png, jpg, jpeg, webp, gif
```

## 五、上传密码

如果 Worker 设置了 `UPLOAD_SECRET`，上传页需要填写同样的密码。

如果 Worker 没有设置 `UPLOAD_SECRET`，密码栏留空即可。

## 六、查看索引

JSON：

```text
https://ayayadaze.github.io/ImageHost/data/images.json
```

CSV：

```text
https://ayayadaze.github.io/ImageHost/data/images.csv
```
