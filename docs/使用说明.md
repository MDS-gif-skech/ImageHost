# 使用说明

## 一、打开上传入口

访问：

```text
https://github.com/MDS-gif-skech/ImageHost/issues/new/choose
```

## 二、上传图片

1. 选择 `Upload image`。
2. 把图片拖入 issue 文本框，或复制截图后粘贴进去。
3. 等 Markdown 中出现 `![...](https://github.com/user-attachments/assets/...)` 后，点击 `Create`。
4. 等待 GitHub Actions 自动处理。
5. 机器人会在 issue 下回复直接 URL 和 Markdown 写法。

## 三、支持格式

```text
png, jpg, jpeg, webp, gif
```

## 四、查看索引

JSON：

```text
https://mds-gif-skech.github.io/ImageHost/data/images.json
```

CSV：

```text
https://mds-gif-skech.github.io/ImageHost/data/images.csv
```
